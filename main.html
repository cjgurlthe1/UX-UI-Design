UX/UI Design

What is UX/UI Design?


User Experience (UX) design and User Interface (UI) design is the most used terms in the web and app design. Often placed together in a single term, UX/UI design, most people are confused because they're thinking it is the same thing.
What is UX design?
"UX" stands for "user experience". A user's experience of an app is determined by how they use it or ineract with it.

What is UI design?
"UI" stands for "user interface". What the user interface is that the whole graphical layout of the app. It contains the text, buttons or links to click on, slides, images, and text entry field. Anything that the user can use to interact with the app itself.
